# 音效素材

本目录存放环境音（ambient/）和 UI 音效（sfx/）。

## 素材来源

推荐以下免费 CC0 音效网站（免注册、免署名、可商用）：

- **Mixkit**: https://mixkit.co/free-sound-effects/
- **Pixabay Sound Effects**: https://pixabay.com/sound-effects/
- **Freesound**: https://freesound.org/ （筛选 CC0 许可）

## 需要的文件

### 环境音 (ambient/)

| 文件名 | 说明 | 建议搜索词 |
|---|---|---|
| waves.mp3 | 海浪+海风（默认，大部分航海场景） | "ocean waves soft ambience" |
| storm.mp3 | 暴风雨（雅茅斯风暴、ch2 风暴） | "storm wind rain waves" |
| harbor.mp3 | 港口嘈杂（赫尔港、巴西港） | "harbor port ambience" |
| ship_creak.mp3 | 船舱内部，木质吱嘎 | "wooden ship creak interior" |
| island.mp3 | 荒岛海滩，海浪+鸟鸣 | "island beach birds waves" |
| silence.mp3 | 寂静/风（回忆、独白场景） | "soft wind silence ambience" |

### UI 音效 (sfx/)

| 文件名 | 说明 | 建议搜索词 |
|---|---|---|
| page_turn.mp3 | 翻页声 | "page turn paper" |
| choice.mp3 | 选项点击 | "soft click select" |
| menu_open.mp3 | 菜单开合 | "menu open close" |
| death.mp3 | 死亡 | "low impact dramatic" |
| ending.mp3 | 结局 | "gentle chime resolve" |

## 格式要求

- MP3 格式，128kbps 以上
- 环境音建议 30 秒以上（循环播放）
- UI 音效建议 0.5-2 秒
- 建议用 Audacity 做 normalize + trim 边缘 + fade in/out

## 占位文件

当前目录下的 .gitkeep 为占位文件。音效下载后替换即可。
游戏引擎在缺失音频文件时会静默跳过，不影响运行。
