### 新建文件
- `src/engine/AudioManager.ts` — Web Audio API 封装
- `public/audio/README.md` — 素材下载说明

### 修改文件
| 文件 | 改动 |
|---|---|
| `src/types/scene.ts` | 新增 `AudioSpec { ambient?, sfx? }`，Scene 加 `audio?` |
| `src/engine/schema.ts` | Zod 同步新增 audio 验证 |
| `src/store/settingsStore.ts` | 新增 `sfxVolume` + `setSfxVolume` |
| `src/ui/SettingsPanel.tsx` | 主音量+音效音量双滑条，删除"预留"文字 |
| `src/store/gameStore.ts` | `_audio` 实例化；`init()` 首次点击解锁 AudioContext；`enterScene()` 切换 ambient + sfx；`chooseChoice()` 播放点击音效 |
| `ch1.json` | 6 个场景分配 audio.ambient（silence/harbor/storm/ship_creak） |
| `ch2.json` | 6 个场景分配 audio.ambient（silence/harbor/storm/ship_creak/island） |

### 架构要点
- **默认 ambient = `"waves"`**（海浪+海风），未显式指定的场景自动使用
- **交叉渐变**：场景切换时旧音轨 1s fade-out + 新音轨 1s fade-in
- **缺失文件静默跳过**：`AudioManager.preload()` catch 后 console.warn，不阻塞游戏
- **AudioContext 延迟解锁**：首次 click/keydown 时 `audio.init()`

### 素材获取
12 个环境音场景（非 waves）+ 5 个 UI 音效。下载源：Mixkit、Pixabay、Freesound（CC0）。详见 `public/audio/README.md`。

## Phase 2 AI 插图 — ⏳ 待实施

### 2.1 火山引擎 Seedream 接入

```
npm install @volcengine/ark-runtime
```

新建 `src/engine/IllustrationGenerator.ts`：
- 封装 `generateImage(prompt, size)` → `{ url, base64 }`
- 模型：`doubao-seedream-5-0-260128`（最新 Lite 版）
- 定价：约 0.2 元/张，新用户 500 张免费
- API Key 通过环境变量 `ARK_API_KEY` 或 `.env` 文件注入

API 端点：
```
POST https://ark.cn-beijing.volces.com/api/v3/images/generations
```

### 2.2 试点场景（6 张）

| 场景 ID | 情感 | size | position |
|---|---|---|---|
| ch1_1_父训 | 压抑，父亲书房 | portrait_4_3 | inline |
| ch1_5_风暴 | 恐怖，巨浪，沉船 | landscape_16_9 | top |
| ch1_8_岸上 | 海滩十字路口 | landscape_16_9 | top |
| ch2_8_猎狮 | 狮子卧礁，紧张 | portrait_4_3 | inline |
| ch2_11_葡船 | 大船放下小艇，救赎 | landscape_16_9 | top |
| ch2_16_荒岛 | 孤身跪沙滩，孤寂 | portrait_4_3 | fullpage |

### 2.3 构建脚本

`scripts/generate-illustrations.ts`：
1. 扫描所有章节 JSON，收集 `illustration.prompt`
2. 跳过 `cached` 已有值的
3. 调用 Seedream API 逐张生成
4. 将 base64 写入 `cached` 字段
5. 手动运行，非运行时生成

### 2.4 渲染

`BookShell.tsx` 场景正文前渲染 `<img>`：
- `top` → 正文上方，最大宽 100%
- `inline` → 正文旁，最大宽 50%，float
- `fullpage` → 全幅背景图
- 移动端全部降级为 top

### 2.5 已有基础

`IllustrationSpec` 类型（`src/types/scene.ts:126`）：
```ts
{
  prompt: string;      // AI 提示词
  alt: string;         // 替代文本
  size: 'square_hd' | 'square' | 'portrait_4_3' | 'portrait_16_9' | 'landscape_4_3' | 'landscape_16_9';
  position: 'top' | 'inline' | 'fullpage';
  cached?: string;     // 生成后的 base64/URL
}
```

Zod schema 已同步（`src/engine/schema.ts:73`），Scene 接口已包含 `illustration?` 字段。
